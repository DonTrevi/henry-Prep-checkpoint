## Explica los siguentes conceptos con tus propias palabras. (no más de tres líneas por respuesta).

* HTML: es un lenguaje de programación que se utiliza exclusivo para diseño de páginas web, básicamente nos ayudará a "armar el esqueleto" de nuesta página.

* CSS: o 'Cascading StyleSheets' es un concepto que se utiliza para referirse a un lenguaje empleado en el diseño gráfico, se usa especialmente en diseño web cuando los sitios se hayan escritos en XML o HTML.

* Git: Git es un software de control de versiones cuyo proposito es llevar registro de los cambios en archivos de computadora y coordinar el trabajo simultaneamente de varias personas trabajando en conjunto sobre un mismo archivo compartido.

* Arreglos: Los arreglos son objetos tipo lista en JavaScript, pueden almacenar dentro de ellos todo tipo de elementos de JavaScript, números, booleanos, funciones y hasta inclusive otros arrays [o arreglos]. Son modificables en todo momento.

* Objetos: Es un contenedor de datos con mútiples elementos relacionados entre sí que pueden ser modificados cuando nosotros queramos.

* Variables: Una variable es, simple y llanamente una forma de almacenar el valor de algo para usar más tarde.

* Métodos: Los métodos son las funciones que pertenecen a un objeto.

* DOM: El Modelo de Objetos del Documento (DOM), es una interfaz de programación de aplicaciones para documentos HTML y XML.

* Atributo de un tag HTML: Los atributos que un tag reibe en HTML nos sirven para cambiar el comportamiento del mismo o configurarlo. Suelen ser definidos con pares Nombre-Valor. 

* Selector CSS: El selector CSS es el puente que comunica las reglas de estilo que se le asigna al documento, y el documento propiamente dicho. Dichas reglas deben ir en otra "hoja" del documento.

* Un función constructora: cumple la función de ahorrarnos cógigo a la hora de crear muchos objetos, básicamente es una función con una "planilla" con elementos que varios objetos comparten, la invocamos cuando queremos crear un nuevo objeto desde cero con los mismos elementos de la planilla en cuestión.
